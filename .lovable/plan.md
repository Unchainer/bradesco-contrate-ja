

# Landing Page - Corretor Bradesco Seguros 🛡️

## Visão Geral
Landing page de alta conversão para corretor de seguros vinculado à Bradesco Seguros, com visual institucional (vermelho/branco), chatbot com IA, e foco em captação de leads via WhatsApp.

---

## 1. Header / Navegação
- Logo/nome do corretor + identidade visual Bradesco Seguros
- Menu simples: Início, Seguros, Dúvidas, Contato
- **Botão flutuante de WhatsApp** sempre visível (canto inferior direito)

## 2. Hero Section (Seção Principal)
- Título forte de conversão: ex. "Proteja o que é seu. Seguros sob medida com a confiança da Bradesco Seguros."
- Subtítulo com proposta de valor
- CTA principal: "Fale com um Especialista" (link WhatsApp)
- Imagem/ilustração profissional de seguros

## 3. Tipos de Seguros (Cards Visuais)
Cards ilustrados para cada categoria:
- 🏠 **Seguro Residencial** — proteção para casa e apartamento
- 🏢 **Seguro Empresarial** — proteção para negócios e comércios
- 🚗 **Seguro Automobilístico** — proteção veicular completa
- 🔧 **Equipamentos e Instrumentos de Trabalho** — máquinas e ferramentas
- 🍽️ **Ramo Alimentício** — restaurantes, food trucks, indústria
- 💼 **Outros Ramos** — seguro de vida, saúde, etc.

Cada card com breve descrição + botão "Saiba Mais" (abre WhatsApp ou seção de detalhes)

## 4. Como Funciona (Passo a Passo)
Seção visual com 3-4 passos simples:
1. Entre em contato pelo WhatsApp
2. Informe suas necessidades
3. Receba uma proposta personalizada
4. Feche seu seguro anual com tranquilidade

## 5. Seção de Dúvidas com Imagens
- Cards visuais/infográficos para explicar conceitos de seguros para novatos
- Imagens ilustrativas respondendo dúvidas comuns (ex: "O que é franquia?", "Como funciona o sinistro?")
- Design visual e didático

## 6. Chatbot com IA (FAQ Inteligente)
- Botão de chat no canto da tela
- Chatbot com IA real (Lovable AI) para responder dúvidas sobre seguros
- Treinado com contexto de seguros Bradesco: coberturas, processos, documentos necessários
- **Requer Lovable Cloud (backend)**

## 7. Depoimentos / Prova Social
- 3-4 depoimentos de clientes satisfeitos
- Estrelas de avaliação
- Selo de parceiro Bradesco Seguros

## 8. CTA Final de Conversão
- Seção chamativa: "Não deixe para depois. Proteja-se agora!"
- Botão grande para WhatsApp
- Formulário simples opcional (nome + telefone + tipo de seguro)

## 9. Footer
- Dados do corretor
- Links úteis
- Redes sociais
- Aviso legal / SUSEP

## 10. SEO Otimizado
- Meta tags otimizadas para: seguro residencial, seguro empresarial, seguro auto, seguro alimentício, corretor Bradesco Seguros
- Estrutura semântica (H1, H2, H3)
- Schema markup para negócio local
- Textos otimizados com palavras-chave do setor de seguros

---

## Requisitos Técnicos
- **Lovable Cloud** para backend do chatbot com IA
- **Botão WhatsApp flutuante** em todas as páginas
- **Design responsivo** (mobile-first)
- **Estilo institucional Bradesco** (vermelho, branco, cinza)
- **Performance otimizada** para SEO
